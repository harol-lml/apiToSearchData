# Docker fastapi

Is a little project to search data using fastapi and docker

## Installation Docker
[Insall](https://www.docker.com/) docker and docker-compose

## Usage

1. Build image

```bash
 docker build -t fastapi:0.01 .
```
2. Run image
```bash
 docker-compose build
 docker-compose up
```
